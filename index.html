Cuestionario de HTML y CSS

Comprueba tus conocimientos sobre los lenguajes de la Web con estas 10 preguntas. Al finalizar obtendrás el resultado y las respuestas correctas.

Comienza ahora

1. ¿Cuál es el lenguaje estándar específico para aplicar estilos de presentación a nuestras páginas web?

  1. Flash>
  2. Javascript
  3. CSS

2. ¿Por qué es conveniente incluir la declaración de DOCTYPE en la página HTML?

  1. Si no se incluye el navegador no puede renderizar (mostrar) la página
  2. Para evitar que el navegador entre en 'Quirk Mode' y la interprete/muestre mal
  3. Para indicar si se ha utilizado o no Flash

3. ¿Para que sirve el atributo 'class'?

  1. Para aplicar unos determinados estilos a los elementos que tenga la misma clase, es decir, el mismo valor en ese atributo
  2. Es específico para indicar el color de la fuente que queremos aplicar, Por ejemplo: p class="red" presentaría las letras en rojo de ese párrafo
  3. Para clasificar el tipo de enlace ('a') que estamos definiendo: 'salto' interno, vínculo a página en el mismo dominio o a un sitio web externo

4. ¿Qué está mal en esta regla de estilo?: .cuadro { border: 1px blue dotted padding: 10px 5px; }

  1. Falta una ',' (coma) entre los dos valores de la propiedad padding (10px 5px)
  2. Falta un ';' (punto y coma) al final de la declaración del estilo 'border'
  3. No se puede poner un '.' (punto) al inicio de una declaración (antes de la palabra 'cuadro)'

5. ¿Cuál crees que es la mejor manera de aplicar estilos a una página web?

  1. Incluir los estilos en las etiquetas HTML para que se carguen y ejecuten antes.
  2. Incluirlos en un fichero externo vinculado a ese fichero HTML.
  3. Incluirlos en la sección cabecera ('head') para agruparlos en un mismo sitio en la misma página.

6. ¿Qué función tiene el elemento 'div'?

  1. Es un elemento divisor, y hace que el navegador muestre una línea horizontal de separación (por ejemplo entre secciones de la página)
  2. Es un contenedor. Crea bloques, por ejemplo para diferenciar distintas secciones de una página (cabecera, pié, columnas..)
  3. Permite realizar una operación arirmética de división en los valores númericos de una tabla (también se utiliza 'sum' y 'dif' para suma y resta)

7. ¿Qué significa que una etiqueta/elemento está 'deprecated' en HTML?

  1. Para aplicar unos determinados estilos a los elementos que tenga la misma clase, es decir, el mismo valor en ese atributoQue si la utilizamos no la entenderán (y no se podrá visualizar bien) en los navegadores
  2. Que ya no forma parte de las especificaciones de la esa versión concreta del estándar HTML
  3. Que ha cambiado de nombre ese elemento en la siguiente versión de HTML

8. Si incluimos una etiqueta de este tipo en nuestra página HTML, ¿qué queremos que haga el navegador?: img src="../icono.gif">

  1. Que incluya la imagen 'icono.gif' que se encuentra en el directorio padre de la carpeta donde está nuestra página HTML
  2. Que incluya la imagen 'icono.gif' que se encuentra en la misma carpeta donde está nuestra página HTML
  3. Que incluya la imagen 'icono.gif' que se encuentra en el directorio 'img'

9. ¿Cómo se incluye un comentario nuestro en una página HTML para que no se interprete por el navegador?

  1. Con 3 asteriscos al principio y final del comentario. Ej: ***comentario sobre el código***
  2. Con 3 barras' al principio y final del comentario. Ej: ///comentario sobre el código///
  3. Con la etiqueta de apertura ' al principio y final del comentario. Ej: /*(!--comentario sobre el código--)*/

10. Para poder modificar el fondo en el cuerpo de una pagina "body" se utiliza:

  1. bgcolor
  2. background
  3. Ninguna de las anteriores
